# Git Commands Practiced

## Initialize Git
```bash
git init
```

## Check repository status
```bash
git status
```

## Add files
```bash
git add .
```

## Commit changes
```bash
git commit -m "commit message"
```

## Push changes
```bash
git push
```

## View commit history
```bash
git log --oneline
```

## Create branch
```bash
git checkout -b learning-branch
```

## Switch branch
```bash
git checkout main
```

## Merge branch
```bash
git merge learning-branch
```
